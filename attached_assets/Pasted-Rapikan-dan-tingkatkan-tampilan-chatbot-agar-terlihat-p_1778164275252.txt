Rapikan dan tingkatkan tampilan chatbot agar terlihat premium, profesional, dan modern. Gunakan HANYA CSS. JANGAN ubah HTML, class/ID, atau JavaScript/fungsi yang sudah berjalan.

🎯 PERBAIKAN HEADER (PENTING):
- Gunakan display: flex, align-items: center, justify-content: space-between pada container header
- Teks "CST Logistics Assistant": 
  • font-size: 16px (mobile) / 18px (desktop)
  • font-weight: 600, line-height: 1.3, letter-spacing: 0.3px
  • color: #ffffff, white-space: nowrap (atau wrap rapi dengan max-width)
  • Pastikan sejajar horizontal dengan icon di sampingnya
- Atur jarak antar icon (speaker, reset, close): gap: 8px (desktop) / 6px (mobile)
- Tambahkan padding header: 14px 16px

✨ VISUAL PREMIUM & ELEGAN:
- Container chat: border-radius: 20px, box-shadow: 0 12px 40px rgba(0,0,0,0.12), 0 4px 12px rgba(0,0,0,0.06)
- Background header: linear-gradient(135deg, #0052D4, #4364F7) atau warna brand CST Logistics
- Chat bubble: border-radius: 16px, padding: 12px 16px, max-width: 85%, shadow: 0 2px 8px rgba(0,0,0,0.06)
- Area input: background: #f8f9fa, border: 1px solid #e9ecef, border-radius: 24px, padding: 10px 16px
- Font global: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif (clean & modern)
- Transisi: transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1) pada tombol & input

📱 RESPONSIVE SEMPURNA:
- @media (max-width: 768px): 
  • Header font-size: 15px, gap icon: 6px, padding header: 12px 14px
  • Container width: 92%, max-width: 380px, height: 70vh
  • Touch target min: 44px, icon size: 20px
- @media (min-width: 769px):
  • Header font-size: 18px, gap icon: 10px, padding header: 16px 20px
  • Container max-width: 450px, height: 600px
- Pastikan tidak ada text overflow, icon tumpang tindih, atau layout pecah di layar kecil

⛔ ATURAN MUTLAK:
- HANYA tambahkan/perbarui CSS styling & layout

- JANGAN hapus fitur (kirim, tracking, form, reset, suara, toggle)
- Pastikan semua interaksi tetap berfungsi 100% seperti sebelumnya
