Any Supabase bearer user is auto-provisioned as an ERP admin
Critical
Authentication and Privilege Escalation
Any user who can obtain a valid Supabase access token is automatically turned into a full admin for the internal ERP API. This allows a normal portal or mobile user to reach admin-only business settings, accounting, reporting, correspondence, and AI management endpoints.

authMiddleware is applied globally before the entire /api router in artifacts/api-server/src/app.ts:55 and accepts any valid Supabase bearer token in artifacts/api-server/src/middlewares/authMiddleware.ts:54-57. When that token belongs to an email address that does not already exist in usersTable, the middleware auto-creates a staff row. The role assignment is fatally wrong at artifacts/api-server/src/middlewares/authMiddleware.ts:75-90: role: isAdmin ? "admin" : "admin". Because both branches are admin, every newly seen bearer-token user is inserted as an admin, not as the schema default of ecommerce.

That admin row is then trusted by requireAdmin in artifacts/api-server/src/lib/requireAdmin.ts:46-53, which loads the user from usersTable and only checks u.role === "admin". Since the middleware has already inserted the row and populated req.user, the very first authenticated bearer-token request can immediately pass admin authorization. The affected admin routes are mounted from artifacts/api-server/src/routes/index.ts and include endpoints such as /api/settings/*, /api/accounting/*, /api/reports/*, /api/payments/*, /api/sales/*, /api/purchase/*, /api/correspondences/*, /api/scan-document/*, and /api/ai-agent/*.

An attacker does not need a staff account. They can register a normal Supabase-backed portal/mobile user, obtain the access token that the application already uses for customer flows, and send that token to an ERP admin endpoint such as /api/settings/notifications. On first use, the API will create an admin record for them and authorize the request. This is a full privilege escalation from low privilege to internal admin control across production business systems.
