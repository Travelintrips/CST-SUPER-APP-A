Tolong perbaiki tampilan image/card pada halaman “Produk Kami” dan “Layanan Kami”.

Masalah saat ini:

* Semua card produk dan layanan masih kosong / tidak memiliki gambar.
* Banyak image preview tidak muncul.
* Tampilan terlihat kurang premium dan kurang memiliki nilai jual.
* Card terlihat generik karena tidak ada visual yang mewakili kategori/folder.

Yang saya inginkan:
Setiap folder/kategori/card harus memiliki gambar HD berkualitas tinggi yang relevan dengan isi folder atau nama layanan/produk.

Contoh:
HALAMAN PRODUK:

* Elektronik → gambar produk elektronik modern
* Furniture → gambar meja/kursi premium
* Kopi / Green Bean → gambar biji kopi HD
* Textile → gambar kain/textile premium
* Alat Tulis → gambar stationery modern

HALAMAN LAYANAN:

* Trucking & Container → gambar truck + container logistics HD
* Freight Laut → kapal cargo/container ship
* Freight Udara → pesawat cargo
* Handling Cargo → warehouse handling/logistics
* Pengurusan Pabean → customs/document handling
* Storage → gudang/logistic warehouse modern

Ketentuan image:

* Gunakan image HD berkualitas tinggi.
* Gunakan aspect ratio konsisten.
* Pastikan image tampil sempurna (tidak broken).
* Gunakan object-cover agar gambar proporsional.
* Tambahkan overlay gradient premium jika perlu.
* Tambahkan hover zoom effect modern.
* Gunakan lazy loading untuk optimasi performa.
* Tambahkan fallback image default jika image gagal load.

UI yang diinginkan:

* Premium SaaS style
* Modern marketplace look
* Visual lebih eksklusif dan profesional
* Card terlihat memiliki nilai jual tinggi
* Responsive desktop & mobile
* Rounded-xl / rounded-2xl
* Soft shadow
* Smooth hover transition
* Elegant image presentation

Penting:

* Jangan menghapus fungsi existing.
* Jangan ubah route/API/logic pemesanan.
* Fokus pada peningkatan visual image/card.

Teknis:

* Pastikan path image valid.
* Pastikan semua gambar bisa tampil untuk:

  * gambar lama
  * gambar baru upload
* Gunakan helper image yang benar:
  asset()
  Storage::url()

Output yang saya harapkan:

* Semua card memiliki gambar HD sesuai kategori/folder.
* Tidak ada broken image.
* Tampilan lebih premium dan profesional.
* Produk & layanan terlihat lebih menarik dan memiliki nilai jual tinggi.
