Buat navbar menjadi super clean, minimal, premium, tapi tetap kuat branding.

TARGET:
Navbar harus terlihat lega, modern, elegan, tidak padat, dan tetap profesional untuk CST Logistics.

PENTING:
- Jangan mengubah bentuk logo.
- Jangan mengubah file logo.
- Jangan stretch/crop logo.
- Jangan merusak routing, language selector, cart, login, register, mega menu services, kalkulator, atau fitur lain.

LAYOUT DESKTOP:
Kiri:
- Logo + “CST Logistics”

Tengah:
- Beranda
- Produk
- Services ▼
- Kalkulator

Kanan:
- Cart icon
- Language selector compact
- Tombol “Masuk”
- Tombol utama “Daftar”

PINDAHKAN KE DROPDOWN “Lainnya” ATAU FOOTER:
- Tentang Kami
- Kontak
- Lacak Pesanan

Jika tetap ingin tampil di navbar, jadikan dropdown kecil “Lainnya”.

DESAIN:
- Navbar height: 72px–80px
- Background: rgba(255,255,255,0.92)
- backdrop-filter: blur(16px)
- border-bottom: 1px solid #E2E8F0
- box-shadow: 0 8px 30px rgba(15,23,42,0.05)

MENU:
- Font size 15px–16px
- Font weight 600
- Normal color: #64748B
- Active color: #0284C7
- Active background: rgba(14,165,233,0.10)
- Border radius: 14px
- Padding menu: 10px 14px
- Gap menu: 18px–24px
- Semua menu harus satu baris.

BRAND:
- Logo height: 42px–46px
- Brand text: “CST Logistics”
- Font weight: 800
- Color: #0F172A
- Jangan pakai nama panjang di navbar agar tidak padat.

KALKULATOR:
- Buat menu Kalkulator sedikit menonjol tapi tetap minimal.
- Gunakan soft pill:
  background rgba(14,165,233,0.08)
  color #0284C7
  border-radius 14px

RIGHT ACTION:
- Cart icon simple.
- Language selector compact: globe + ID/EN, jangan tampil double.
- Tombol Masuk dibuat plain text.
- Tombol Daftar dibuat dark premium:
  background #0F172A
  color white
  border-radius 16px
  padding 12px 18px

SERVICES MEGA MENU:
- Tetap ada.
- Desain clean, tidak terlalu besar.
- Maksimal lebar 720px–900px.
- Grid 2 kolom atau 3 kolom.
- Gunakan icon kecil, judul, deskripsi singkat.
- Jangan membuat navbar terlihat ramai.

MOBILE:
- Logo + CST Logistics di kiri.
- Hamburger di kanan.
- Semua menu masuk drawer.
- Drawer clean, accordion untuk Services.
- Kalkulator tetap mudah terlihat.
- Jangan overflow.
- Jangan menutupi logo.

I18N:

- Jangan hardcode teks.
