Tambahkan fitur suara/notifikasi audio pada chatbot untuk pengalaman lebih interaktif di mobile (HP), dengan ketentuan:

🔊 JENIS SUARA YANG DITAMBAHKAN:
- ✅ Pesan terkirim: "ting" lembut (short positive chime)
- ✅ Bot membalas: "pop" atau "blip" halus (different pitch dari kirim)
- ✅ Notifikasi baru: "ding" singkat (saat ada pesan masuk saat chatbot tertutup)
- ✅ Error/gagal kirim: "buzz" atau tone rendah (sekali saja, tidak mengganggu)

📱 KHUSUS MOBILE/HP:
- Gunakan Web Audio API atau <audio> preload="auto" untuk kompatibilitas iOS & Android
- Audio hanya aktif setelah user pertama kali berinteraksi (tap/klik) — hindari autoplay blocked
- Volume default: 30-40% (tidak kagetkan user), tetap hormati silent mode HP
- Format audio: .mp3 atau .ogg, durasi < 0.5 detik, ukuran file < 10KB per suara

🎨 UX & AKSESIBILITAS:
- Tambahkan tombol mute/unmute kecil di header chatbot (icon: 🔊/🔇)
- Simpan preferensi user di localStorage (jika user mute, tetap mute di sesi berikutnya)
- Berikan visual feedback juga (animasi kecil) sebagai alternatif jika suara dimatikan
- Pastikan suara tidak mengganggu saat user sedang telepon atau pakai headset

⚡ IMPLEMENTASI TEKNIS (CSS/JS ONLY):
- Buat fungsi playSound(type) yang dipanggil saat:
  • user kirim pesan → playSound('sent')
  • bot kirim balasan → playSound('received')  
  • pesan error → playSound('error')
- Gunakan AudioContext atau <audio hidden> dengan ID unik per suara
- Tambahkan class .sound-enabled / .sound-muted di container untuk styling toggle button

⛔ ATURAN WAJIB:
- JANGAN ubah fungsi chatbot yang sudah ada (kirim, tracking, form, toggle)
- JANGAN paksa autoplay suara tanpa interaksi user pertama
- JANGAN gunakan audio besar/berat yang memperlambat loading mobile
- HANYA tambahkan fitur suara sebagai enhancement opsional

🎯 TUJUAN:
Pengguna mobile merasa lebih "hidup" dan engaged saat berinteraksi dengan chatbot, tanpa mengganggu atau memperlambat pengalaman. Suara memberi konfirmasi instan bahwa aksi mereka berhasil — meningkatkan kepercayaan dan nilai premium.
