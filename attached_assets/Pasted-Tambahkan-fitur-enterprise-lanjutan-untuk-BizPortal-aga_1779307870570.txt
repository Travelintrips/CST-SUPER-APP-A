Tambahkan fitur enterprise lanjutan untuk BizPortal agar sistem menjadi logistics ERP + RFQ platform modern dengan mini form architecture.

PENTING:
- Semua flow tetap menggunakan MINI FORM TERPISAH
- Dashboard utama hanya untuk management/admin
- Semua mini form mobile-first, cepat, ringan, dan mudah dibuka dari WhatsApp

==================================================
1. VENDOR PERFORMANCE & RATING SYSTEM
==================================================

Tambahkan Vendor Performance System.

Sistem harus menyimpan dan menghitung:
- on-time delivery percentage
- average response speed
- cancel rate
- POD upload completeness
- ETA accuracy
- customer complaint count
- order success rate
- average rating

Tampilkan rating vendor di:
- Admin Vendor Comparison
- Vendor List
- Order Detail

Contoh tampilan:
⭐ 4.9
On-time: 98%
Avg response: 12 menit
Completed Orders: 125

Tambahkan badge:
- Top Vendor
- Fast Response
- Best ETA
- Trusted Vendor

Tambahkan tabel:
vendor_performance:
- id
- vendor_id
- total_orders
- completed_orders
- cancelled_orders
- ontime_percentage
- average_response_minutes
- pod_completeness_score
- eta_accuracy_score
- customer_rating
- updated_at

4. POD OCR & AUTO VERIFICATION
==================================================

Saat vendor/driver upload POD:
- gunakan OCR untuk membaca dokumen

OCR harus mencoba membaca:
- nomor surat jalan
- nomor order
- tanggal
- penerima
- tanda tangan
- nama perusahaan

Sistem harus:
- mencocokkan dengan order
- memberi warning jika data tidak sesuai

Tambahkan:
pod_ocr_results:
- id
- order_id
- extracted_text
- extracted_order_number
- extracted_date
- extracted_receiver
- verification_status
- confidence_score
- created_at

Tampilkan:
- OCR success
- OCR failed
- verification mismatch
