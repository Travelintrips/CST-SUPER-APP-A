Tolong upgrade header/hero halaman Produk agar lebih profesional dan memiliki background image yang bisa diedit/diganti.

MASALAH:
- Area biru terlalu polos.
- Tidak ada gambar background.
- Teks “Katalog Produk”, “Produk Kami”, deskripsi, dan search input kurang menonjol.

TARGET:
Buat hero Produk lebih premium dengan background image sesuai halaman produk/logistik, namun tetap mudah dibaca.

PERBAIKAN:
1. Tambahkan background image di belakang overlay biru.
   Tema gambar yang cocok:
   - warehouse storage
   - logistics products
   - cargo inventory
   - product catalog warehouse
   - supply chain warehouse

2. Background image harus mudah diganti.
   Buat variable/config, contoh:
   const productHeroBg = "/images/product-hero.jpg";

   Atau CSS variable:
   --product-hero-bg: url("/images/product-hero.jpg");

3. Jangan buat biru solid.
   Gunakan gradient overlay transparan agar gambar tetap terlihat.

Contoh style:
background:
  linear-gradient(
    135deg,
    rgba(14,165,233,0.82),
    rgba(37,99,235,0.58)
  ),
  url("/images/product-hero.jpg") center/cover no-repeat;

4. Tambahkan overlay gelap tipis agar teks jelas:
text-shadow: 0 4px 18px rgba(15,23,42,0.25);

5. Perjelas typography:
- “Katalog Produk”:
  font-size 13px–14px
  font-weight 800
  letter-spacing 0.14em
  text-transform uppercase
  color putih / cyan sangat terang

- “Produk Kami”:
  font-size clamp(42px, 5vw, 72px)
  font-weight 800
  line-height 1.05
  color white
  text-shadow halus

- Deskripsi:
  font-size 18px–21px
  color rgba(255,255,255,0.9)
  max-width 680px

6. Search input:
- Buat lebih premium dan jelas.
- Background rgba(255,255,255,0.16)
- Border 1px solid rgba(255,255,255,0.35)
- Backdrop-filter blur(12px)
- Border-radius 16px
- Placeholder lebih jelas
- Icon search tetap rapi
- Focus state biru/putih dengan shadow halus

7. Layout:
- Content max-width 720px.
- Padding atas/bawah lebih proporsional.
- Mobile responsive.
- Jangan overflow.

8. I18N:
Pastikan semua teks masuk translation:
productsHero.label
productsHero.title
productsHero.description
productsHero.searchPlaceholder

9. Jangan Merusak:
- Jangan ubah navbar.
- Jangan ubah logo.
- Jangan merusak filter kategori produk.
- Jangan merusak search logic.
- Jangan merusak routing/API/backend.

HASIL:
Header Produk terlihat lebih modern, profesional, punya background image yang bisa diganti, dan teks/search lebih jelas.
