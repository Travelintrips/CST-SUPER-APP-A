🔴 ATURAN BISNIS (WAJIB DIHAFAL)
1. MODE TRANSPORTASI:
   - TRUCKING: TIDAK pakai ETA/ETD. Hanya `pickup_date` & `pickup_time`. Rute: Kota + Kecamatan.
   - AIR/SEA: WAJIB pakai `etd` & `eta`. Rute: Kota/Pelabuhan. Field: weight, volume, incoterm.
2. ESTIMASI HARGA (Form Customer):
   - TRUCKING: HANYA hitung dari vendor dengan `year_vehicle >= (TahunSekarang - 5)`.
   - AIR/SEA: Hitung dari semua vendor aktif. TIDAK filter tahun.
3. ANONIMISASI VENDOR (PENTING):
   - Customer TIDAK BOLEH melihat nama/vendor asli di manapun (Web form, Halaman Pilih Opsi, WA, atau Invoice awal).
   - Tampilkan label generik: "Opsi 1", "Opsi 2", dst. Hanya tampilkan: Harga, Tahun Unit/Transit Time, Tipe Unit/Carrier, Jadwal (Pickup/ETD-ETA).
   - Nama vendor HANYA terlihat di Admin Dashboard & internal database untuk akuntansi/SO.
4. FLOW: Customer Request → Admin Terima → Admin Pilih Vendor Terbaik → Kirim Opsi (Tanpa Nama) ke Customer → Customer Pilih 1 → Generate SO (Admin link internal ke vendor).
