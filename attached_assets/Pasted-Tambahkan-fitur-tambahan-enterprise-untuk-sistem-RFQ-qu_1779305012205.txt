Tambahkan fitur tambahan enterprise untuk sistem RFQ, quotation, vendor comparison, dan order execution pada BizPortal.

PENTING:
Semua fitur harus tetap menggunakan konsep MINI FORM TERPISAH dan dashboard utama hanya untuk admin management.

========================================
1. RESPONSE DEADLINE / AUTO EXPIRED
========================================

Tambahkan response deadline pada RFQ vendor.

Saat admin blast RFQ ke vendor:
- admin dapat menentukan deadline response
- contoh: 30 menit, 2 jam, 1 hari, custom datetime

Vendor mini form harus menampilkan:
- countdown timer
- waktu sisa response

Jika vendor belum submit sampai deadline:
- ubah status menjadi expired
- nonaktifkan tombol submit
- tampilkan pesan:
  "Penawaran sudah expired"

Tambahkan field:
rfqs:
- response_deadline

vendor_responses:
- expired_at

rfq_vendor_links:
- expired_at

========================================
2. AUTO VENDOR RANKING
========================================

Tambahkan sistem ranking vendor otomatis pada Admin Vendor Comparison.

Sistem harus menghitung:
- harga termurah
- ETA tercepat
- rating vendor terbaik
- response tercepat
- margin terbaik

Tampilkan badge:
- Best Price
- Fastest ETA
- Best Rating
- Recommended Vendor

Tambahkan score otomatis:
vendor_score =
(price_weight + eta_weight + rating_weight + response_weight)

Admin tetap bisa memilih vendor manual meskipun bukan ranking pertama.

========================================
3. AUTO MARGIN SYSTEM
========================================

Tambahkan sistem margin otomatis.

Saat admin memilih vendor:
- sistem otomatis menghitung harga jual customer

Contoh:
customer_price =
vendor_price + margin

Margin bisa:
- fixed amount
- percentage
- minimum margin

Tambahkan setting:
margin_rules:
- service_type
- route
- customer_type
- minimum_margin
- margin_type
- margin_value

Contoh:
- trucking = 15%
- air freight = 10%
- PPJK = fixed Rp500.000

Admin tetap dapat override harga manual.

========================================
4. QUOTATION PDF GENERATOR
========================================

Saat admin klik:
"Kirim Penawaran ke Customer"

Sistem otomatis generate PDF quotation profesional.

PDF harus berisi:
- logo perusahaan CST
- quotation number
- customer name
- service type
- route
- cargo detail
- ETA
- terms & conditions
- harga final
- valid until
- catatan
- signature placeholder

Generate PDF:
- mobile friendly
- printable
- professional corporate layout

Simpan:
quotation_pdf_url

Customer dapat:
- lihat PDF
- download PDF

========================================
5. CUSTOMER QUOTE VALIDITY
========================================

Tambahkan validity/expiry pada quotation customer.

Contoh:
"Penawaran berlaku sampai 21 Mei 2026 17:00"

Jika expired:
- customer tidak bisa approve
- status = quotation_expired

Tambahkan:
customer_quote_links:
- valid_until

========================================
6. AUDIT TRAIL / ACTIVITY LOG
========================================

Tambahkan audit trail lengkap.

Catat:
- siapa buat RFQ
- siapa blast vendor
- siapa pilih vendor
- siapa ubah harga
- siapa approve customer
- siapa update order
- siapa upload POD
- siapa buat invoice

Tambahkan tabel:
activity_logs:
- id
- rfq_id
- order_id
- actor_type
- actor_id
- action
- old_value
- new_value
- created_at

========================================

Tetapi:
- RFQ number baru
- token baru
- status reset
