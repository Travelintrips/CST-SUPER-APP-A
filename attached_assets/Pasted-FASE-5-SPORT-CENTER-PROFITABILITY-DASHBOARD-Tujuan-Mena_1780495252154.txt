FASE 5 — SPORT CENTER PROFITABILITY DASHBOARD

Tujuan:

* Menampilkan laba/rugi Sport Center berbasis data Accounting yang sudah terposting.
* Mendukung multi-company (khusus PT Cahaya Sejati Teknologi).
* Menampilkan profit per fasilitas, per periode, dan per jenis pendapatan.

A. BACKEND API

Buat endpoint baru:

GET /api/sport-center/profitability

Parameter:

* company_id
* from
* to
* facility_id (optional)

Response:

{
revenue_booking: number,
revenue_membership: number,
refund_amount: number,
operational_expense: number,
gross_profit: number,
net_profit: number,
bookings_count: number,
active_members: number,
top_facilities: []
}

B. SUMBER DATA

Revenue Booking:

source = 'sport_center_booking'

Revenue Membership:

source = 'sport_center_membership'

Refund:

source IN (
'sport_center_refund',
'sport_center_booking_refund',
'sport_center_booking_reversal'
)

Operational Expense:

source = 'sport_center_operational_expense'

Semua query wajib filter:

company_id
