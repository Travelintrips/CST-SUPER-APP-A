TECHNICAL REQUIREMENT:
1. Buat seed/default data untuk semua template di database atau constants file.
2. Jika template belum pernah disimpan user, tampilkan default preset.
3. Jika user sudah edit dan save template, gunakan template user.
4. Tambahkan tombol "Reset ke Default Template".
5. Jangan biarkan textarea kosong untuk workflow utama.
6. Pastikan preview langsung menampilkan template default.
7. Pastikan conditional block tetap bisa dirender.
