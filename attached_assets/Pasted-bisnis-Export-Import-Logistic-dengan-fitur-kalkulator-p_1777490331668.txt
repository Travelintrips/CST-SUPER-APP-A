bisnis Export Import & Logistic dengan fitur kalkulator pemesanan step by step.

Nama aplikasi:
Export Import & Logistic Ordering System

Style desain:
- Modern, clean, corporate, enterprise
- Warna utama biru navy, putih, abu muda
- Responsive desktop dan mobile
- Gunakan card layout, icon profesional, smooth animation
- Cocok untuk perusahaan freight forwarding, customs clearance, trucking, warehouse, dan trading export-import

Tech stack:
- React / Next.js
- Tailwind CSS
- Supabase sebagai database
- Gunakan state sementara untuk cart/order sebelum submit ke database

==================================================
HALAMAN UTAMA
==================================================

Buat landing page dengan section:
1. Hero Section
   - Judul: Export Import & Logistic Solutions
   - Subtitle: Freight, Customs Clearance, Trucking, Warehousing & End-to-End Logistics Services
   - CTA Button: Start Booking / Get Estimate

2. Service Flow Section
   Jelaskan bahwa customer dapat memilih tipe pengiriman, memilih layanan, menghitung estimasi biaya, lalu submit order.

3. Booking Calculator Section
   Ini adalah fitur utama.

==================================================
FLOW BOOKING CALCULATOR
==================================================

Step 1:
Customer pilih tipe pengiriman:
- Import
- Export
- Domestic
- Door to Door

Gunakan card besar dengan icon dan deskripsi singkat.

Step 2:
Setelah tipe pengiriman dipilih, tampilkan kategori jasa dalam bentuk StatsCard:

Kategori:
1. Freight
2. Customs
3. Handling
4. Storage
5. Trucking
6. Document
7. Additional

Setiap StatsCard harus menampilkan:
- Icon/logo kategori
- Nama kategori
- Deskripsi pendek
- Total item jasa dalam kategori
- Button: View Services

Step 3:
Saat StatsCard diklik, tampilkan detail kategori:
- Icon/logo besar
- Nama kategori
- Deskripsi lengkap
- Button kembali ke kategori
- Daftar item jasa sesuai kategori

Contoh item jasa:

Freight:
- Air Freight
- Sea Freight FCL
- Sea Freight LCL
- Domestic Freight

Customs:
- Import Customs Clearance
- Export Customs Clearance
- PIB/PEB Handling
- HS Code Consultation

Handling:
- Origin Handling
- Destination Handling
- Loading / Unloading
- DG Cargo Handling

Storage:
- Warehouse Storage
- Temporary Storage
- Bonded Warehouse
- Cold Storage

Trucking:
- Pickup Trucking
- Delivery Trucking
- Container Trucking
- Cargo Trucking

Document:
- Shipping Instruction
- Bill of Lading
- Air Waybill
- Certificate of Origin
- Packing List / Invoice Review

Additional:
- Insurance
- Surveyor
- Permit Assistance
- Quarantine / BPOM / SNI Support
