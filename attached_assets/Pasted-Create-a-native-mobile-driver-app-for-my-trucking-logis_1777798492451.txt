Create a native mobile driver app for my trucking logistics system using React Native + Expo in Replit.

This app will be used by truck drivers for trucking shipment operations and must connect to my existing web app backend/API/database.

App name: CST Driver

Main goals:
1. Driver can login securely.
2. Driver can see assigned trucking jobs.
3. Driver can accept or reject a job.
4. Driver can update shipment status step by step.
5. Driver can upload proof photos and documents.
6. Driver can send GPS location updates.
7. Admin web dashboard can receive driver status updates in real time or near real time.

Core screens:
- Splash screen
- Driver login
- Home dashboard
- Assigned jobs list
- Job detail page
- Start pickup button
- Pickup confirmation page
- Upload photo/document page
- In-transit tracking page
- Delivery confirmation page
- Digital signature / Proof of Delivery page
- Completed jobs history
- Profile page

Shipment status flow:
ASSIGNED
ACCEPTED
ON_THE_WAY_TO_PICKUP
ARRIVED_AT_PICKUP
PICKED_UP
IN_TRANSIT
ARRIVED_AT_DESTINATION
DELIVERED
COMPLETED
CANCELLED

Each job should display:
- Job number
- Customer name
- Pickup address
- Delivery address
- Cargo description
- Vehicle type
- Truck plate number
- Driver name
- Pickup date/time
- Delivery date/time
- Special instruction
- Current shipment status
