Saya ingin membangun ulang modul Sea Freight menjadi modul baru bernama Ocean Freight dengan flow lengkap seperti aplikasi freight forwarding modern.

Jangan buat halaman detail layanan statis lagi.

Tujuan modul:
Customer bisa cek estimasi Ocean Freight berdasarkan rute, shipment type, container/cargo, dan layanan tambahan.
Harga awal bersifat estimasi.
Setelah inquiry masuk, admin bisa blast RFQ ke vendor / shipping line / NVOCC / co-loader / forwarder partner.
Vendor/partner submit rate.
Admin compare rate, pilih rate final, tambah markup/PPN, lalu kirim quote final ke customer.
Customer menerima notifikasi, review harga final, lalu approve.
Setelah approve, admin confirm booking dan update tracking shipment.

Gunakan nama module:
Ocean Freight

Service yang didukung:
- FCL
- LCL
- Domestic Sea
- Export
- Import
- Cross Border
- Port to Port
- Door to Port
- Port to Door
- Door to Door

FASE 8 — PRICING ENGINE OCEAN FREIGHT
==================================================

Buat helper function:
calculateOceanFreightEstimate(input, rates)

Input:
- origin_city
- origin_port
- destination_city
- destination_port
- trade_type
- shipment_type
- service_mode
- container_type
- container_qty
- total_cbm
- gross_weight
- cargo_condition
- selected_additional_services
- currency_display

Rules:

1. Ambil rate aktif yang cocok:
- origin_port
- destination_port
- shipment_type
- service_mode
- container_type jika FCL
- valid_from <= today
- valid_until >= today
- is_active = true

2. Formula FCL:
base = ocean_freight_amount * container_qty

3. Formula LCL:
chargeable_cbm = MAX(total_cbm, lcl_minimum_cbm)
base = chargeable_cbm * lcl_rate_per_cbm

4. Standard charges:
charges =
thc_origin
+ thc_destination
+ doc_fee
+ bl_fee
+ do_fee
+ handling_fee

5. Additional service charges:
Jika dipilih:
+ customs_clearance_fee
+ trucking_pickup_estimate
+ trucking_delivery_estimate

6. Cargo surcharge:
Jika cargo_condition = DG Cargo:
tambahkan dg_surcharge_percent
Jika cargo_condition = Reefer:
tambahkan reefer_surcharge
Jika ada peak season:
tambahkan peak_season_surcharge

7. Currency:
Jika currency bukan IDR:
- tampilkan total dalam currency asli
- tampilkan equivalent IDR dengan exchange_rate_to_idr

8. Return 3 opsi estimasi:
- Economy: harga termurah
- Standard: kombinasi harga dan transit terbaik
- Priority: transit tercepat

Return breakdown:
- estimate_option
- rate_id
- rate_source_name
- carrier
- route
- shipment_type
- service_mode
- container_type
- transit_days
- direct_or_transshipment
- base_ocean_freight
- origin_charges
- destination_charges
- document_charges
- trucking_charges
- customs_charges
- surcharge_breakdown
- total_estimate
- currency
- total_estimate_idr
- price_status
- validity
