Tolong redesign tampilan website agar lebih modern, profesional, elegan, dan konsisten di semua halaman.

FOKUS UTAMA:
1. Desain ulang navbar/header navigation menjadi lebih modern.
2. Tambahkan glass effect, shadow halus, spacing yang rapi, hover animation, active state yang jelas.
3. Buat navbar responsive untuk mobile/tablet.
4. Buat UI system/style guide agar semua halaman konsisten.
5. JANGAN mengubah bentuk logo, file logo, proporsi logo, atau desain logo. Logo boleh hanya disesuaikan ukuran, spacing, alignment, dan posisi agar terlihat rapi.

DETAIL NAVBAR:
- Gunakan layout:
  kiri: logo + nama perusahaan
  tengah: menu navigasi
  kanan: cart, admin, dashboard, tombol keluar
- Background navbar:
  putih semi-transparan / glassmorphism
  backdrop blur
  border bawah tipis
  shadow halus
- Navbar harus sticky di atas halaman.
- Active menu seperti “Beranda” dibuat lebih elegan, bukan border hitam tebal.
- Gunakan active state berupa background biru muda lembut + teks biru.
- Tambahkan hover underline animasi atau background soft.
- Tombol “Keluar” dibuat modern dengan rounded corner, hover effect, dan icon tetap rapi.
- Jarak antar menu dibuat konsisten dan tidak terlalu rapat.

RESPONSIVE MOBILE:
- Pada layar kecil, menu tengah dan kanan disembunyikan ke hamburger menu.
- Saat hamburger diklik, tampil mobile menu/dropdown/drawer.
- Mobile menu harus rapi, mudah diklik, dan tidak menutupi logo.
- Logo tetap tampil di kiri.
- Jangan merusak bentuk logo pada mobile.

UI SYSTEM:
Buat atau rapikan global style/theme dengan token konsisten:
- Font utama: Inter, Poppins, atau Manrope.
- Primary color: #0EA5E9 atau #2563EB.
- Dark color: #0F172A.
- Text color: #1E293B.
- Muted text: #64748B.
- Background: #F8FAFC.
- Border: #E2E8F0.
- Radius: 12px, 16px, 20px.
- Shadow:
  soft shadow untuk card/navbar/button.
- Spacing system:
  8px, 12px, 16px, 24px, 32px, 48px.

KONSISTENSI HALAMAN:
Terapkan style yang sama ke:
- Beranda
- Produk
- Jasa/Services
- Tentang Kami
- Kontak
- Lacak Pesanan
- Dashboard Admin
- Cart/Checkout jika ada

KOMPONEN YANG HARUS DIRAPIKAN:
- Navbar
- Button primary dan secondary
- Card produk/layanan
- Form input
- Section title
- Badge/status
- Table dashboard admin
- Modal jika ada

HERO SECTION:
- Jangan mengubah logo.
- Gunakan desain hero yang lebih premium:
  background image/logistik dengan overlay gradient gelap.
- Teks harus mudah dibaca.
- CTA button dibuat lebih standout.
- Layout desktop boleh center atau kiri, tapi harus terlihat modern.
- Mobile harus readable dan tidak overflow.

ANIMASI:
Tambahkan transisi ringan:
- hover menu
- hover button
- hover card
- active state
Gunakan animasi halus 0.2s–0.3s, jangan berlebihan.

PERFORMA:
- Jangan menambahkan library berat jika tidak perlu.
- Pastikan tidak merusak fungsi login, dashboard, cart, admin, routing, dan fitur yang sudah ada.
- Pastikan tidak mengubah struktur data/API/backend.

OUTPUT YANG SAYA MAU:
1. Update desain navbar modern.
2. Responsive mobile navbar dengan hamburger.
3. Global UI system/theme.
4. Semua halaman terlihat konsisten.
5. Logo tetap sama bentuknya dan tidak dimodifikasi.
6. Pastikan tidak ada error.
7. Setelah selesai, jelaskan file apa saja yang diubah.
