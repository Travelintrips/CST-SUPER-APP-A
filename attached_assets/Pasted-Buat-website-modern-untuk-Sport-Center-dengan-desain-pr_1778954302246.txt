==================================================
TUJUAN WEBSITE
==================================================

Website ini digunakan untuk menampilkan fasilitas sport center dan menyediakan sistem booking online untuk lapangan atau kelas olahraga.

==================================================
FITUR UTAMA
==================================================

1. LANDING PAGE
Buat landing page dengan struktur:

- Navbar sticky
- Logo / nama Sport Center
- Menu navigasi:
  - Home
  - Fasilitas
  - Jadwal
  - Booking
  - Tentang Kami
  - Kontak

Hero section:
- Judul besar:
  "Sport Center Bandara Soekarno Hatta"
- Subtitle:
  "Tempat terbaik untuk olahraga, komunitas, dan gaya hidup sehat."
- CTA button:
  "Booking Sekarang"
  "Lihat Fasilitas"

Tambahkan background image sport center / olahraga dengan overlay modern.

==================================================

2. DAFTAR FASILITAS OLAHRAGA
Buat section fasilitas olahraga dalam card modern.

Contoh fasilitas:
- Lapangan Futsal
- Lapangan Badminton
- Lapangan Basket
- Lapangan Tenis
- Fitness Area
- Kelas Yoga
- Kelas Zumba
- Cafe & Lounge

Setiap card berisi:
- gambar
- nama fasilitas
- deskripsi singkat
- harga mulai dari
- tombol booking

==================================================

3. JADWAL KELAS / LAPANGAN
Buat halaman atau section jadwal.

Tampilkan jadwal dalam bentuk tabel/card:
- Hari
- Jam
- Jenis aktivitas
- Lokasi/lapangan
- Slot tersedia
- Harga
- Tombol booking

Contoh:
Senin - Jumat:
06.00 - 08.00 Futsal
08.00 - 10.00 Badminton
17.00 - 19.00 Basket
19.00 - 21.00 Zumba

==================================================

4. SISTEM BOOKING ONLINE
Buat halaman booking/reservasi online.

Form booking:
- Nama lengkap
- Nomor WhatsApp
- Email
- Pilih fasilitas
- Pilih tanggal
- Pilih jam
- Jumlah peserta
- Catatan tambahan
- Metode pembayaran:
  - Bayar di tempat
  - Transfer bank
  - QRIS

Setelah submit:
- tampilkan halaman/alert konfirmasi booking
- generate kode booking otomatis
- tampilkan ringkasan booking
- status booking default: "Menunggu Konfirmasi"

Booking tidak perlu payment gateway sungguhan, cukup simulasi UI.

==================================================

5. DASHBOARD ADMIN SEDERHANA
Buat halaman admin sederhana untuk melihat data booking.

Route:
/admin

Fitur admin:
- daftar booking
- filter berdasarkan tanggal
- filter berdasarkan status
- ubah status booking:
  - Menunggu Konfirmasi
  - Dikonfirmasi
  - Selesai
  - Dibatalkan

Gunakan data dummy/local state/localStorage jika belum memakai database.
