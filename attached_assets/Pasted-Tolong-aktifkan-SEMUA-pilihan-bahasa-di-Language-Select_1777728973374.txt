Tolong aktifkan SEMUA pilihan bahasa di Language Selector, bukan hanya ID/EN.

TARGET:
Semua bahasa yang ada di dropdown harus benar-benar aktif dan menerjemahkan seluruh website secara global.

WAJIB DIDUKUNG:
- id-ID: Bahasa Indonesia
- en-US: English
- en-GB: English
- zh-CN: 中文（简体）
- zh-TW: 中文（繁體）
- ja-JP: 日本語
- ko-KR: 한국어
- ms-MY: Bahasa Melayu
- ar-AE: العربية
- ar-SA: العربية
- de-DE: Deutsch
- fr-FR: Français
- nl-NL: Nederlands
- es-ES: Español
- it-IT: Italiano
- hi-IN: हिन्दी
- en-AU: English

TUGAS:
1. Lengkapi translations.js untuk semua locale di atas.
2. Semua halaman harus menggunakan t("key"), bukan teks hardcoded.
3. Semua teks yang terlihat user harus diterjemahkan:
   - navbar
   - hero
   - produk
   - layanan
   - tentang kami
   - kontak
   - lacak pesanan
   - cart/checkout
   - login/register
   - dashboard admin
   - footer
   - tombol
   - label form
   - placeholder
   - error/success message
   - table header
   - status pesanan
   - modal/dialog

JANGAN:
- Jangan mengubah bentuk logo.
- Jangan mengubah file logo.
- Jangan merusak desain navbar.
- Jangan merusak routing, login, cart, dashboard, API, atau backend.

IMPLEMENTASI:
Jika project React:
- Gunakan LanguageProvider global.
- Gunakan hook useLanguage().
- Semua komponen ambil teks dari t("key").
- LanguageSelector memanggil setLanguage(locale).
- Simpan pilihan bahasa ke localStorage dengan key "app_language".
- Saat refresh, bahasa tetap aktif.
- Saat pindah halaman, bahasa tetap aktif.
- Saat pilih bahasa, semua teks langsung berubah tanpa reload.

FALLBACK:
- Jika satu key belum ada di bahasa tertentu, fallback ke en-US.
- Jika en-US tidak ada, fallback ke id-ID.
- Jangan pernah tampilkan key mentah seperti "about.title" ke user.

ARABIC / RTL:
Untuk bahasa Arabic:
- Set document.documentElement.dir = "rtl".
- Set document.documentElement.lang sesuai locale.
Untuk bahasa lain:
- Set dir = "ltr".

AKURASI:
Terjemahan harus natural, profesional, dan sesuai konteks logistik, bukan literal mentah.

QUALITY CHECK:
Setelah selesai:
1. Coba pilih setiap bahasa satu per satu.
2. Pastikan seluruh halaman berubah bahasa.
3. Pastikan dropdown checkmark mengikuti bahasa aktif.
4. Pastikan refresh tidak mengembalikan bahasa ke default.
5. Pastikan pindah halaman tetap pakai bahasa yang dipilih.
6. Pastikan Arabic tampil RTL.
7. Pastikan tidak ada error console.
8. Pastikan tidak ada teks hardcoded yang masih tertinggal.
