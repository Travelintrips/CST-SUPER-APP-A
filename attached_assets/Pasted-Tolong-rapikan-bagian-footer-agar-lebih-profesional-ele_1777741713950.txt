Tolong rapikan bagian footer agar lebih profesional, elegan, modern, dan friendly.

MASALAH SAAT INI:
- Logo dan teks brand di footer terlihat terlalu besar/kurang seimbang.
- Nama perusahaan turun terlalu banyak baris.
- Spacing antar kolom masih bisa dirapikan.
- Footer terlihat terlalu polos dan kurang premium.
- Informasi kontak perlu dibuat lebih clean dan mudah dibaca.

TARGET:
Footer harus terlihat premium, rapi, clean, dan konsisten dengan navbar serta UI system website.

PERBAIKAN FOOTER:
1. Layout
- Gunakan container max-width 1200px–1440px.
- Footer dibagi menjadi 4 kolom:
  1) Brand
  2) Tautan Cepat
  3) Layanan
  4) Hubungi Kami
- Gunakan grid responsive:
  desktop: 4 kolom
  tablet: 2 kolom
  mobile: 1 kolom

2. Brand Area
- Jangan mengubah bentuk logo.
- Jangan mengubah file logo.
- Jangan crop/stretch logo.
- Ukuran logo sekitar 48px–56px.
- Logo dan teks brand sejajar rapi.
- Jika nama terlalu panjang, boleh gunakan:
  “CST Logistics”
  atau tetap “PT. Cahaya Sejati Teknologi” dengan ukuran font lebih proporsional.
- Deskripsi brand dibuat lebih rapi:
  “Solusi logistik terintegrasi dan berbasis teknologi untuk kebutuhan ekspor, impor, dan distribusi bisnis Anda.”

3. Desain Footer
Gunakan style lebih premium:
- Background boleh tetap biru, tapi buat lebih elegan dengan gradient:
  linear-gradient(135deg, #0284C7, #0EA5E9)
- Tambahkan pattern/overlay sangat halus jika cocok.
- Text putih dengan opacity berbeda:
  heading: white
  link/description: rgba(255,255,255,0.78)
- Border top atau divider halus:
  rgba(255,255,255,0.18)

4. Link Footer
- Link diberi hover effect:
  warna lebih putih
  geser sedikit ke kanan
  transition 0.2s
- Spacing antar link konsisten 10px–12px.
- Jangan terlalu rapat.

5. Kontak
Rapikan kontak dengan icon konsisten:
- Lokasi
- Telepon / WhatsApp
- Email

Setiap kontak:
- icon ukuran 20px
- teks sejajar vertikal tengah
- gap 12px
- clickable:
  alamat bisa link Google Maps jika tersedia
  telepon pakai tel:
  email pakai mailto:
  WhatsApp pakai wa.me

6. Copyright
- Buat copyright berada di bawah footer.
- Center aligned.
- Divider di atas copyright.
- Font size 14px.
- Warna rgba(255,255,255,0.65).

7. Floating WhatsApp
- Tombol WhatsApp di kanan bawah jangan menutupi konten footer.
- Tambahkan margin bottom/right yang cukup.
- Pastikan responsive.
- Hover effect smooth.
- Di mobile tetap nyaman diklik.

8. Language Support
Semua teks footer wajib masuk i18n:
footer.description
footer.quickLinks
footer.home
footer.services
footer.customerPortal
footer.servicesTitle
footer.seaFreight
footer.airFreight
footer.customsBrokerage
footer.domesticDistribution
footer.contactUs
footer.location
footer.phone
footer.email
footer.copyright

Jangan hardcode teks.

9. Jangan Merusak
- Jangan mengubah logo.
- Jangan merusak navbar.
- Jangan merusak routing.
- Jangan merusak fitur WhatsApp.
- Jangan merusak i18n/language selector.

OUTPUT:
Setelah selesai, jelaskan file apa saja yang diubah dan pastikan tidak ada error console.
