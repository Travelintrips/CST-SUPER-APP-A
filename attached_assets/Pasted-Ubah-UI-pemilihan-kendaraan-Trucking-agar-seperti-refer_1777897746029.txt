Ubah UI pemilihan kendaraan Trucking agar seperti referensi gambar, tetapi desain disesuaikan dengan branding CST Logistics.

PENTING:
- Jangan ubah layout besar halaman.
- Jangan ubah logic kalkulasi utama.
- Jangan ubah halaman lain.
- Fokus hanya pada Vehicle Type di halaman Trucking.

TARGET:
Vehicle Type dibuat lebih visual dan mudah dipilih, bukan dropdown teks biasa.

DESAIN:
- Gunakan style CST Logistics:
  - biru utama #0EA5E9 / #0284C7
  - navy #0F172A
  - putih / light background
  - border radius modern
  - shadow halus
- Jangan pakai warna hijau seperti referensi.
- Jangan terlihat seperti copy paste aplikasi lain.
- Harus clean, premium, friendly, dan cocok untuk website CST Logistics.

IMPLEMENTASI:

1. Ganti dropdown Vehicle Type menjadi card/select panel visual.

Daftar kendaraan:
- CDE
- CDD
- Fuso
- Wingbox
- Trailer Truck

Setiap item tampil dengan:
- icon kendaraan kecil
- nama kendaraan
- hover state
- selected state warna biru soft

2. Untuk Trailer Truck:
Saat user memilih Trailer Truck, tampilkan pilihan ukuran langsung di bawahnya dalam bentuk segmented control/tab:

- Trailer 20 ft
- Trailer 40 ft
- Trailer Flatbed

Contoh tampilan:
[ Trailer 20 ft ] [ Trailer 40 ft ]
[ Trailer Flatbed ]

3. Selected state:
- Background biru CST
- Text putih
- Border biru
- Shadow halus

4. Jika selain Trailer dipilih:
- Jangan tampilkan pilihan ukuran trailer.

5. Data yang disimpan:
Jika user pilih CDE:
vehicleType = "CDE"
trailerSize = null

Jika user pilih Trailer 20 ft:
vehicleType = "Trailer Truck"
trailerSize = "20 ft"

Jika user pilih Trailer 40 ft:
vehicleType = "Trailer Truck"
trailerSize = "40 ft"

Jika user pilih Trailer Flatbed:
vehicleType = "Trailer Truck"
trailerSize = "Flatbed"

6. Validasi:
- Vehicle Type wajib dipilih.
- Jika Vehicle Type = Trailer Truck, trailerSize wajib dipilih.
- Tombol Tambahkan ke Pesanan disabled jika belum lengkap.

7. Ringkasan/keranjang:
Tampilkan:
- CDE
- CDD
- Fuso
- Wingbox
- Trailer Truck - 20 ft
- Trailer Truck - 40 ft
- Trailer Truck - Flatbed

8. Jangan gunakan submenu dropdown tambahan.
   Buat pengalaman seperti gambar referensi: pilih kendaraan → jika Trailer, muncul opsi ukuran.

HASIL:
Pemilihan kendaraan Trucking terlihat lebih modern, visual, mudah dipakai, dan punya nilai jual tinggi sesuai brand CST Logistics.
