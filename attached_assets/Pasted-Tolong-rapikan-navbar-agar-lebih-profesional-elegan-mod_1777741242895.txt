Tolong rapikan navbar agar lebih profesional, elegan, modern, dan friendly.

MASALAH SAAT INI:
- Logo dan nama perusahaan terlalu besar sehingga navbar terasa penuh.
- Teks “PT. Cahaya Sejati Teknologi” turun ke 2 baris dan membuat navbar terlalu tinggi.
- Spacing antar menu belum seimbang.
- Beberapa menu seperti “Tentang Kami” dan “Lacak Pesanan” turun ke 2 baris.
- Navbar terlihat padat dan kurang premium.

TARGET DESAIN:
- Navbar terlihat clean, rapi, premium, dan mudah dibaca.
- Tinggi navbar ideal sekitar 76px–88px di desktop.
- Semua menu utama tampil 1 baris di desktop.
- Logo tetap tidak berubah bentuk, tidak crop, tidak stretch.

PERBAIKAN WAJIB:
1. Logo & Brand
- Jangan mengubah file logo.
- Jangan mengubah bentuk logo.
- Atur ukuran logo sekitar 42px–48px.
- Teks brand jangan terlalu besar.
- Gunakan font-weight 700/800.
- Jika teks terlalu panjang, gunakan:
  “CST Logistics”
  atau tetap “PT. Cahaya Sejati Teknologi” tapi pastikan 1 baris.
- Logo dan teks sejajar vertikal di tengah.

2. Layout Navbar
- Gunakan flex layout yang rapi:
  kiri: logo + brand
  tengah: menu utama
  kanan: cart, language selector, login, register
- Gunakan max-width container, misalnya 1440px.
- Navbar full width, tapi isi navbar berada di container tengah.
- Gunakan justify-between dan gap konsisten.

3. Menu Utama
Menu desktop:
- Beranda
- Produk
- Jasa/Services
- Kalkulator
- Tentang Kami
- Kontak
- Lacak Pesanan

Semua menu harus 1 baris.
Gunakan white-space: nowrap.

4. Active State
- Jangan pakai style terlalu besar.
- Active menu gunakan background biru muda lembut.
- Border-radius 14px–16px.
- Teks active warna #0EA5E9 atau #0284C7.

5. Hover State
- Tambahkan hover background soft.
- Tambahkan transition 0.2s–0.3s.
- Jangan berlebihan.

6. Right Actions
- Cart icon, language selector, login, dan register harus sejajar.
- Register button dibuat premium:
  background #0F172A
  color white
  border-radius 14px–16px
  padding 12px 18px
- Login tampil simple dan clean.
- Language selector dibuat compact:
  icon globe + kode bahasa
  contoh: ID
  jangan tampil “ID ID” dua kali.
- Cart icon diberi ukuran konsisten.

7. Glass / Premium Effect
Navbar gunakan:
- background: rgba(255,255,255,0.9)
- backdrop-filter: blur(16px)
- border-bottom: 1px solid #E2E8F0
- box-shadow: 0 8px 30px rgba(15,23,42,0.06)

8. Responsive Mobile
- Di tablet/mobile, menu tengah masuk hamburger menu.
- Logo tetap tampil rapi di kiri.
- Jangan sampai brand turun berantakan.
- Jika perlu, pada mobile gunakan brand pendek “CST Logistics”.
- Hamburger menu kanan.
- Mobile drawer/accordion harus rapi.
- Jasa/Services tetap bisa accordion di mobile.
- Kalkulator tetap muncul di mobile menu.

9. Jangan Merusak
- Jangan ubah bentuk logo.
- Jangan ubah file logo.
- Jangan merusak mega menu Jasa/Services.
- Jangan merusak Language Selector.
- Jangan merusak login, cart, dashboard, register, routing, API, backend.

OUTPUT:
Setelah selesai, jelaskan file apa saja yang diubah dan pastikan tidak ada error console.
