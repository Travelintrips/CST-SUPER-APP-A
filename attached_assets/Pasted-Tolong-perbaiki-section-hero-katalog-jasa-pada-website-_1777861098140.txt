Tolong perbaiki section hero/katalog jasa pada website CST Logistics agar terlihat lebih profesional, modern, dan memiliki nilai jual visual (branding logistik & global).

TARGET:
- Warna biru lebih elegan, tidak terlalu terang, tetap fresh & friendly.
- Teks lebih jelas, kontras nyaman di mata.
- Tambahkan background visual yang relevan dengan logistik (global, pengiriman, konektivitas) tapi tetap subtle & tidak ramai.
- Kesan: corporate, international logistics, premium, terpercaya.

IMPLEMENTASI:

1. Background utama (gradient modern):
.hero {
  position: relative;
  overflow: hidden;
  background: linear-gradient(135deg, #0B4F8A 0%, #0F8FD8 50%, #38BDF8 100%);
}

2. Tambahkan overlay gelap lembut agar teks lebih readable:
.hero::after {
  content: "";
  position: absolute;
  inset: 0;
  background: linear-gradient(
    90deg,
    rgba(2, 34, 64, 0.55) 0%,
    rgba(2, 34, 64, 0.25) 40%,
    rgba(2, 34, 64, 0.05) 100%
  );
  pointer-events: none;
}

3. Tambahkan elemen visual bernilai jual (pilih salah satu atau kombinasi halus):

OPSIONAL A (disarankan - clean & modern):
Tambahkan pattern garis/globe network:
.hero::before {
  content: "";
  position: absolute;
  inset: 0;
  background-image: radial-gradient(rgba(255,255,255,0.15) 1px, transparent 1px);
  background-size: 40px 40px;
  opacity: 0.25;
}

OPSIONAL B (lebih premium):
Tambahkan SVG atau image transparan:
- ilustrasi peta dunia (world map)
- garis koneksi global (network lines)
- siluet kontainer / kapal / pesawat (opacity rendah)

Contoh:
background: url('/images/world-map.svg') no-repeat right center;
background-size: 60%;
opacity: 0.15;

4. Konten hero harus di atas semua layer:
.hero-content {
  position: relative;
  z-index: 2;
}

5. Styling teks:
- KATALOG JASA:
  color: rgba(255,255,255,0.85);
  font-weight: 600;
  letter-spacing: 0.12em;

- Layanan Kami:
  color: #ffffff;
  font-weight: 800;
  text-shadow: 0 6px 20px rgba(0,0,0,0.25);

- Deskripsi:
  color: rgba(255,255,255,0.9);
  font-size: 20–24px;
  line-height: 1.6;

6. Search box (lebih modern & glass effect):
.search-box {
  background: rgba(255,255,255,0.15);
  backdrop-filter: blur(12px);
  border: 1px solid rgba(255,255,255,0.35);
  border-radius: 14px;
  box-shadow: 0 10px 30px rgba(0,0,0,0.2);
}

.search-box input::placeholder {
  color: rgba(255,255,255,0.75);
}

.search-box svg {
  color: rgba(255,255,255,0.85);
}

7. Jaga agar:
- Tidak terlalu ramai
- Tetap clean & corporate
- Mobile friendly
- Fokus utama tetap ke teks & search

OUTPUT:
Hasil harus terlihat:
✔ Lebih premium
✔ Lebih “logistics global company”
✔ Lebih enak dibaca
✔ Lebih punya identitas brand
