Tolong buat semua link footer menjadi lebih smooth, profesional, dan elegan saat diklik.

TARGET LINK FOOTER:
Tautan Cepat:
- Beranda
- Tentang Kami
- Kalkulator Biaya
- Lacak Pesanan
- Portal Pelanggan

Layanan Kami:
- Pengiriman Laut Internasional
- Pengiriman Udara
- Bea Cukai & Kepabeanan
- Distribusi Domestik

FUNGSI:
1. Semua link harus aktif dan menuju halaman/route masing-masing.
2. Setelah halaman terbuka, posisi scroll harus otomatis berada di bagian paling atas halaman.
3. Gunakan smooth transition/scroll.
4. Jangan membuka halaman di posisi scroll sebelumnya.

ROUTE:
- Beranda → /
- Tentang Kami → /tentang-kami
- Kalkulator Biaya → /kalkulator
- Lacak Pesanan → /lacak-pesanan
- Portal Pelanggan → /portal-pelanggan

Layanan:
- Pengiriman Laut Internasional → /jasa-services#sea-freight
- Pengiriman Udara → /jasa-services#air-freight
- Bea Cukai & Kepabeanan → /jasa-services#customs-brokerage
- Distribusi Domestik → /jasa-services#domestic-distribution

Jika route di project berbeda, sesuaikan dengan route yang sudah ada.

SMOOTH SCROLL:
- Untuk pindah halaman biasa, setelah route berubah jalankan:
  window.scrollTo({ top: 0, behavior: "smooth" })

- Untuk hash section seperti #sea-freight:
  setelah route terbuka, scroll ke section tersebut dengan:
  element.scrollIntoView({ behavior: "smooth", block: "start" })

- Jika section belum ada, buat id section sesuai:
  id="sea-freight"
  id="air-freight"
  id="customs-brokerage"
  id="domestic-distribution"

REACT ROUTER:
Jika menggunakan React Router:
- Gunakan <Link> atau <NavLink>, bukan div clickable.
- Buat komponen global ScrollToTop.
- ScrollToTop harus listen location.pathname dan location.hash.
- Jika location.hash ada, scroll ke element hash.
- Jika tidak ada hash, scroll ke top.

DESAIN LINK FOOTER:
Buat hover/click lebih premium:
- transition: all 0.25s ease
- hover color: #FFFFFF
- hover transform: translateX(4px)
- active/focus ring jelas
- cursor pointer
- jangan terlalu mencolok

AKSESIBILITAS:
- Semua link harus bisa diakses keyboard.
- Gunakan aria-label jika perlu.
- Focus ring terlihat.

I18N:
Pastikan label footer tetap memakai t("key"), jangan hardcode.

JANGAN MERUSAK:
- Jangan ubah logo.
- Jangan merusak footer layout.
- Jangan merusak WhatsApp floating button.
- Jangan merusak tombol Back to Top.
- Jangan merusak navbar, language selector, cart, login, dashboard, API, atau backend.

OUTPUT:
Setelah selesai jelaskan file apa saja yang diubah dan pastikan tidak ada error console.
