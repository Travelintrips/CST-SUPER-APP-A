==================================================
FLOW BOOKING CALCULATOR
==================================================

Step 1:
Customer pilih tipe pengiriman:
- Import
- Export
- Domestic
- Door to Door

Gunakan card besar dengan icon dan deskripsi singkat.

Step 2:
Setelah tipe pengiriman dipilih, tampilkan kategori jasa dalam bentuk StatsCard:

Kategori:
1. Freight
2. Customs
3. Handling
4. Storage
5. Trucking
6. Document
7. Additional

Setiap StatsCard harus menampilkan:
- Icon/logo kategori
- Nama kategori
- Deskripsi pendek
- Total item jasa dalam kategori
- Button: View Services

Step 3:
Saat StatsCard diklik, tampilkan detail kategori:
- Icon/logo besar
- Nama kategori
- Deskripsi lengkap
- Button kembali ke kategori
- Daftar item jasa sesuai kategori

Contoh item jasa:

Freight:
- Air Freight
- Sea Freight FCL
- Sea Freight LCL
- Domestic Freight

Customs:
- Import Customs Clearance
- Export Customs Clearance
- PIB/PEB Handling
- HS Code Consultation

Handling:
- Origin Handling
- Destination Handling
- Loading / Unloading
- DG Cargo Handling

Storage:
- Warehouse Storage
- Temporary Storage
- Bonded Warehouse
- Cold Storage

Trucking:
- Pickup Trucking
- Delivery Trucking
- Container Trucking
- Cargo Trucking

Document:
- Shipping Instruction
- Bill of Lading
- Air Waybill
- Certificate of Origin
- Packing List / Invoice Review

Additional:
- Insurance
- Surveyor
- Permit Assistance
- Quarantine / BPOM / SNI Support

Step 4:
Saat item jasa diklik, tampilkan kalkulator khusus item tersebut.

Contoh kalkulator:

Air Freight:
Input:
- Origin Airport
- Destination Airport
- Gross Weight
- Volume Length, Width, Height
- Quantity
- Rate per Kg
Formula:
Volume Weight = Length x Width x Height x Quantity / 6000
Chargeable Weight = max(Gross Weight, Volume Weight)
Total = Chargeable Weight x Rate per Kg

Sea Freight FCL:
Input:
- Origin Port
- Destination Port
- Container Type: 20FT / 40FT / 40HC
- Freight Rate
- Handling Fee
Total = Freight Rate + Handling Fee

Sea Freight LCL:
Input:
- CBM
- Weight
- Rate per CBM
- Minimum Charge
Total = max(CBM x Rate per CBM, Minimum Charge)

Customs Clearance:
Input:
- Shipment Type: Import / Export
- Customs Service Fee
- Document Fee
- PIB/PEB Fee
- Additional Permit Fee
Total = semua biaya dijumlahkan

Trucking:
Input:
- Pickup City
- Destination City
- Vehicle Type: CDE, CDD, Fuso, Wingbox, Trailer
- Distance
- Trucking Rate
- Loading Fee
Total = Trucking Rate + Loading Fee

Storage:
Input:
- Number of Days
- CBM / Pallet / KG
- Rate per Day
Total = Days x Quantity x Rate

Document:
Input:
- Document Type
- Quantity
- Fee per Document
Total = Quantity x Fee per Document

Additional:
Input:
- Service Type
- Service Fee
- Admin Fee
Total = Service Fee + Admin Fee

Step 5:
Setelah customer klik button “Add to Order”, item masuk ke Ringkasan Pesanan sementara/cart/state.

Jangan langsung simpan ke database.

Cart harus menampilkan:
- Service Category
- Service Item
- Calculation Detail
- Quantity / Weight / CBM jika ada
- Subtotal
- Button edit
- Button remove

Step 6:
Tampilkan Total Estimasi dari semua item:
- Subtotal
- Admin Fee jika ada
- Tax/VAT optional
- Grand Total Estimate

Tambahkan catatan:
“This is an estimated price. Final quotation will be confirmed by our team.”

Step 7:
Setelah klik “Lanjutkan Pesanan”, tampilkan form customer.

Form Customer:
- Company Name
- Customer Name
- Email
- Phone / WhatsApp
- Shipment Type
- Origin
- Destination
- Commodity
- Cargo Description
- Gross Weight
- Volume / CBM
- Required Date
- Notes
- Upload Document optional

Step 8:
Setelah submit, simpan data ke database Supabase:

Table: orders
Fields:
- id
- order_number
- company_name
- customer_name
- email
- phone
- shipment_type
- origin
- destination
- commodity
- cargo_description
- gross_weight
- volume_cbm
- required_date
- notes
- subtotal
- tax
- grand_total
- status
- created_at

Default status:
New Order

Table: order_items
Fields:
- id
- order_id
- category
- service_name
- calculator_type
- input_data JSONB
- calculation_result JSONB
- subtotal
- created_at

OrderItems harus terhubung dengan Orders melalui order_id.

Step 9:
Setelah order berhasil dibuat:
- Tampilkan halaman sukses
- Tampilkan Order Number
- Tampilkan ringkasan pesanan
- Tampilkan pesan bahwa tim akan menghubungi customer
