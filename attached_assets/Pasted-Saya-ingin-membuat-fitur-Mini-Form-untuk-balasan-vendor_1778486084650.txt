Saya ingin membuat fitur Mini Form untuk balasan vendor/admin ke customer via WhatsApp.

Kebutuhan fitur:

1. Buat halaman/komponen Mini Form Reply untuk Admin dan Vendor.
2. Mini form digunakan untuk membuat balasan quotation ke customer secara rapi, lalu dikirim via WhatsApp melalui integrasi Fonnte yang sudah ada.
3. Form harus memiliki field:
   - Nomor Order / RFQ ID
   - Nama Customer
   - Nomor WhatsApp Customer
   - Jenis Layanan
   - Rute / Tujuan
   - Estimasi Harga Vendor
   - Markup Admin dalam persen atau nominal tetap
   - Harga Final Customer
   - Estimasi Pickup
   - Estimasi Delivery
   - Catatan Vendor/Admin
   - Status: Ready / Need Info / Not Available
4. Jika vendor mengisi harga vendor, sistem otomatis menghitung harga final customer berdasarkan setting markup.
5. Admin harus bisa review dulu sebelum pesan dikirim ke customer.
6. Buat preview pesan WhatsApp sebelum dikirim.
7. Format pesan WhatsApp harus profesional seperti ini:

Halo {customerName},

Berikut update quotation untuk permintaan layanan Anda:

No. RFQ: {rfqId}
Layanan: {serviceType}
Rute: {route}
Estimasi Pickup: {pickupDate}
Estimasi Delivery: {deliveryDate}
Harga: Rp {finalPrice}
Status: {status}

Catatan:
{notes}

Silakan konfirmasi apabila quotation ini disetujui.

Terima kasih,
CST Logistics

8. Tambahkan tombol:
   - Save Draft
   - Preview WhatsApp
   - Send to Customer
   - Send to Admin Group
9. Semua log balasan harus tersimpan di database, termasuk:
   - rfqId
   - vendorId
   - adminId
   - customerPhone
   - vendorPrice
   - markupType
   - markupValue
   - finalPrice
   - messageBody
   - status
   - sentAt
   - createdAt
10. Pastikan tidak merusak fitur WhatsApp/Fonnte yang sudah berjalan.
11. Gunakan struktur project yang sudah ada. Cari dulu file route/API Fonnte existing, order/RFQ existing, dan database schema existing sebelum membuat file baru.
12. Jika table belum ada, buat migration/table baru yang aman.
13. Setelah selesai, jelaskan file apa saja yang diubah dan cara testing dari customer order → vendor reply → admin review → kirim WhatsApp ke customer.
