# PROJECT: Sport Center Bandara Soekarno Hatta

## 🎯 OBJECTIVE
Buat website booking sport center yang modern, responsif, dan profesional dengan sistem reservasi online sederhana.

## ⚙️ TECH STACK (WAJIB)
- Framework: React + Vite + TypeScript
- Styling: TailwindCSS (mobile-first)
- Icons: lucide-react
- State: React hooks + localStorage
- Routing: React Router DOM v6
- Form: React Hook Form + Zod (opsional, bisa pakai native validation)

## 🎨 DESIGN SYSTEM
### Warna Utama:
- primary: #2563EB (biru sporty)
- secondary: #10B981 (hijau fresh)
- accent: #F97316 (oranye energi)
- background: #F8FAFC
- text: #1E293B

### Style Guidelines:
- Rounded corners: rounded-xl untuk card, rounded-full untuk button
- Shadow: shadow-md hover:shadow-lg transition
- Gradient: bg-gradient-to-r dari biru ke hijau untuk hero/CTA
- Font: Inter atau system sans-serif
- Animasi: hover:scale-[1.02] transition-transform duration-200

### Responsive Rules:
- Mobile-first: grid-cols-1 → md:grid-cols-2 → lg:grid-cols-3
- Navbar: desktop = horizontal menu, mobile = hamburger + drawer
- Tabel jadwal: scrollable horizontal di mobile
