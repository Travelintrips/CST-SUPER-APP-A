Tolong aktifkan fitur Language Selector yang sudah tampil di navbar.

MASALAH:
Saat ini dropdown bahasa sudah muncul, tetapi saat memilih bahasa, teks website belum berubah.

TUJUAN:
Saat user memilih bahasa, seluruh teks utama website berubah sesuai bahasa yang dipilih.

WAJIB:
1. Jangan mengubah bentuk logo.
2. Jangan mengubah desain logo.
3. Jangan merusak desain navbar yang sudah ada.
4. Jangan merusak routing, login, cart, dashboard, atau fitur lain.
5. Language Selector tetap berada di navbar kanan pada desktop dan masuk ke mobile menu pada responsive.

FITUR YANG HARUS DIBUAT:
1. Buat sistem i18n sederhana atau gunakan sistem i18n yang sudah ada jika tersedia.
2. Simpan bahasa terpilih ke localStorage.
3. Saat halaman direfresh, bahasa terakhir tetap aktif.
4. Default bahasa adalah id-ID.
5. Jika browser language cocok dengan data bahasa, gunakan browser language sebagai default.
6. Saat memilih bahasa di dropdown, update:
   - label language selector, contoh EN / ID
   - tanda aktif/checkmark
   - semua teks website yang sudah memiliki translation key

IMPLEMENTASI:
Jika project React:
1. Buat LanguageContext atau hook useLanguage.
2. Buat function t(key) untuk mengambil translation.
3. Bungkus App dengan LanguageProvider.
4. Ganti teks hardcoded menjadi t("nama_key").
5. LanguageSelector memanggil setLanguage(locale).
6. Simpan locale ke localStorage.

Contoh pemakaian:
{t("nav_home")}
{t("hero_title")}

Jika project bukan React:
1. Gunakan attribute data-i18n pada elemen teks.
2. Saat bahasa berubah, update semua element [data-i18n].
3. Simpan pilihan ke localStorage.

AKSESIBILITAS:
- Dropdown bisa dibuka dengan keyboard.
- Bisa ditutup dengan Escape.
- Bisa ditutup saat klik di luar.
- Gunakan aria-label="Pilih bahasa".
- Gunakan aria-expanded pada tombol dropdown.
- Focus ring tetap terlihat.

DATA LANGUAGE SELECTOR:
Pastikan language list tetap ada:
- id-ID Bahasa Indonesia
- en-US English
- en-GB English
- zh-CN 中文
- ja-JP 日本語
- ko-KR 한국어
Dan bahasa lain yang sudah ada.

CATATAN:
Untuk bahasa yang belum ada translation lengkap, fallback ke en-US atau id-ID agar tidak error.
Jangan tampilkan key mentah ke user jika translation tidak ditemukan.

OUTPUT:
Setelah selesai, jelaskan:
1. File apa saja yang diubah/dibuat.
2. Cara menambah bahasa baru.
3. Cara menambah teks translation baru.
4. Pastikan tidak ada error di console.
