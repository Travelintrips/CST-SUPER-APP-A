PERBAIKAN KHUSUS SECTION KATALOG JASA (Jasa/Services)

PENTING:
- Kembalikan tampilan Katalog Produk ke kondisi semula (sebelum perubahan styling hero).
- Pastikan tidak ada styling dari Katalog Jasa yang “bocor” ke Katalog Produk.
- Semua perubahan hanya berlaku untuk halaman/section Jasa/Services.
- Jangan ubah layout, struktur, logic, routing, atau komponen lain.

LANGKAH:

1. Scope styling hanya untuk Jasa/Services:
Gunakan class atau wrapper khusus, misalnya:
.page-services .hero { ... }

Pastikan CSS tidak menggunakan selector global seperti:
.hero { ... }  ❌

2. Kembalikan Katalog Produk:
- Hapus atau override semua styling baru yang sebelumnya mempengaruhi hero di halaman produk.
- Pastikan halaman produk menggunakan styling lama (default sebelumnya).

3. Terapkan desain modern hanya di Jasa/Services:
.page-services .hero {
  position: relative;
  overflow: hidden;
  background: linear-gradient(135deg, #0B4F8A 0%, #0F8FD8 50%, #38BDF8 100%);
}

.page-services .hero::after {
  content: "";
  position: absolute;
  inset: 0;
  background: linear-gradient(
    90deg,
    rgba(2, 34, 64, 0.55) 0%,
    rgba(2, 34, 64, 0.25) 40%,
    rgba(2, 34, 64, 0.05) 100%
  );
}

.page-services .hero::before {
  content: "";
  position: absolute;
  inset: 0;
  background-image: radial-gradient(rgba(255,255,255,0.15) 1px, transparent 1px);
  background-size: 40px 40px;
  opacity: 0.25;
}

.page-services .hero-content {
  position: relative;
  z-index: 2;
}

4. Styling teks hanya untuk Jasa:
.page-services .hero h1 {
  color: #ffffff;
  font-weight: 800;
}

.page-services .hero p {
  color: rgba(255,255,255,0.9);
}

5. Search box hanya di Jasa:
.page-services .search-box {
  background: rgba(255,255,255,0.15);
  backdrop-filter: blur(12px);
  border: 1px solid rgba(255,255,255,0.35);
}

OUTPUT:
✔ Katalog Produk kembali seperti semula
✔ Katalog Jasa tampil modern & premium
✔ Tidak ada efek styling silang antar halaman
✔ Clean, rapi, dan terpisah dengan jelas
